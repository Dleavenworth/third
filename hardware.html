    What is a motherboard and what does it do?

    A motherboard allows the diffrent parts to communicate with each other and provides housing for some parts.

    What is a CPU and what does it do?

    CPU stands for central prosesing unit. The CPU directs the entire computer to carry out stored commands but does not execute it itself.

    What is RAM and what does it do?

    RAM stands for random access memory. RAM allocates space for applications and executeables to run and process functions.

    What is a hard drive and what does it do?

    A hard drive stores user data and your OS for the RAM to access and execute.

    How do the diffrent parts communicate with each other?

    Several "bus's" transfer data between the diffrent parts of the computer.

    What are the keyboard mouse and monitor and what do they do?

    The keyboard and mice collect user input and transfer it to the diffrent parts to execute them then the input after being processed then it displays on the monitor.

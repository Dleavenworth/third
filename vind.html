What is vindinium?

Vindinium is an Artificial Intelligence programming challenge. You have to take the control of a legendary hero using the programming language of your choice. You will fight with other AI for a predetermined number of turns and the hero with the greatest amount of gold will win.

How does it work?

Vindinium works by creating a randomized map and sending all the data to your bot in the way of JSON. JSON is JavaScript Object Notation the notation that is used to write very large objects. You might use it if you want to store lots of information in one object. After your bot recives the JSON then it uses it for whatever your bot does with it. Then you respond with the changed JSON objects.

How does my bot work?

My bot uses a very long and basic if/else chain that checks the health of the bot and determines what to do based on the JSON. If the health if below 51 then it moves to a tavern and heals. If the health is above 51 and there are mines for it to take then it takes the mines. If an enemy bot is closer than a mines then it attacks the bot. Else it will sit infront of a tavern when it is winning. Lastly if a direction is failed to be passed before the last else then it is given a randomized direction.
